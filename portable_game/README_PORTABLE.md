# Crimson Eclipse - Portable Game

## How to Run

This folder contains a fully portable version of the Crimson Eclipse browser game. No installation or setup is required.

### Quick Start

1. Simply open `index.html` in any modern web browser
2. The game will run completely in your browser - no internet connection required
3. Use WASD or Arrow Keys to move your character
4. Survive as long as possible!

### System Requirements

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software or dependencies needed
- Works on Windows, Mac, Linux, and mobile devices

### Game Files

- `index.html` - Main game file (open this to play)
- `js/` - All game code and logic
- `design.md` - Game design document
- `devlog.md` - Development progress log
- `CLAUDE.MD` - Project instructions
- `FEEDBACK.MD` - Player feedback

### Moving to Another Computer

To move this game to another computer:
1. Copy this entire `portable_game` folder
2. Transfer it to the new computer using any method (USB drive, cloud storage, etc.)
3. Open `index.html` in a web browser on the new computer
4. The game will work exactly the same way!

### About the Game

Crimson Eclipse is a vampire survivors-style browser game where you survive waves of enemies while collecting upgrades and growing stronger. The game features:

- Fast-paced survival gameplay
- Multiple weapon and upgrade systems
- Progressive difficulty with stage system
- Meta-progression system for long-term advancement
- Retro-styled graphics with a dark horror theme

Enjoy playing Crimson Eclipse!
